Technologies

I am open to all modern DevOps tools. Given a greenfield project and discretion over which technologies could be employed, my choices would include:

  • Infrastructure Automation: Terraform
  • Configuration Management: Ansible
  • Source Mangement: Git
  • Continuous Integration: Jenkins
  • Monitoring: Cloudwatch, CloudTrail or Nagios
  • Collaboration: JIRA
  • Containers: Docker, Swarm or Kubernetes (for larger requirements)
  • Image Repositories: DockerHub, Docker Registry or ECR
  • Messsage Brokering: RabbitMQ, ZeroMQ or AWS SQS
  • Languages: NodeJs, Python, Bash, Make
  • Infrastructure: AWS (EC2, S3, ECR, ECS, RDS, Route53, API Gateway, etc)

Given a brownfield project: I learn existing systems and those targeted for improvement. I remain flexible to legacy or preferred technologies and provide suggestions where improvements might be realized.
